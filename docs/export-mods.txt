R: libdv
S: +

M: export_ffmpeg.c
D: (video) * | (audio) MPEG/AC3/PCM
C: libmp3lame
R: none
S: +
I: Encodes many different formats to both AVI and raw. Supported are mpeg1video, mpeg2video, mpeg4, mjpeg, h263, h263p, wmv1, wmv2, rv10, msmpeg4, msmpeg4v2, huffyuv and dvvideo.

M: export_im.c
D: (video) * | (audio) MPEG/AC3/PCM
C: libImageMagick
R: none
S: +
I: Encodes image sequences by using the ImageMagick library. ImageMagick is able to handle a lot of different image formats among are png, jpg, miff, tiff, etc. Use -F to select the desired format.

M: export_jpg.c
D: (video) * | (audio) MPEG/AC3/PCM
C: libjpeg
R: none
S: +
I: Encodes jpg image sequences using libjpeg. Faster than ImageMagick. Use -F to select the compression quality.

M: export_lame.c
D: (audio) MPEG 1/2
C: libmp3lame
R: lame sox
S: +
I: An audio-only encoder which drives the lame binary. The tool sox is used to do resampling if required. Encodes to a MP3 file.

M: export_lzo.c
D: (video) LZO real-time compression | (audio) MPEG/AC3/PCM
C: liblzo libmp3lame
R: none
S: +
I: Encodes video using a loss-less real-time LZO codec. This codec is a homegrown invention of transcode and is intended as an intermediate storage format. MPlayer can playback LZO-based AVI files as well. 

M: export_mjpeg.c
D: (video) Motion JPEG | (audio) MPEG/AC3/PCM
C: libmp3lame libjpeg
R: none
S: -
I: Encodes MJPEG based AVI files using a homegrown algorithm based on libjpeg. Using ffmpeg -F mjpeg for this task is a good idea.

M: export_mov.c

S: o

M: export_mp2enc.c
D: (audio) MPEG 1/2
C: mjpegtools
R: mp2enc
S: +
I: Drives the mp2enc binary and writes an MP2 (MPEG1-Layer2) file. Useful for when encoding to SVCD to be multiplexed with mplex after encoding.

M: export_mpeg.c
D: (video) MPEG 1/2 | (audio) MPEG 1 Layer II
C: nasm
R: none
S: +
I: Interface to the bbmpeg library (included in transcode). It can encode generic mpeg1, VCD, SVCD, MPEG2 and DVD type video. Encoded video goes into a elementary file to be multiplexed with the corresponding audio file after encoding.

M: export_mpeg2enc.c
D: (video) MPEG 1/2
C: mjpegtools
R: mpeg2enc
S: +
I: Drives the mpeg2enc binary. mpeg2enc is a very feature rich MPEG encoder, have a look at its manpage. Encodes generic mpeg1, VCD, SVCD, MPEG2 and DVD type video.
